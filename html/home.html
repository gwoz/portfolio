Battleground

A crowd gaming iPhone app built in React Native using a Rails API and Postgres geospatial-enabled database. After signing in using Facebook or Google, geolocation data emitted by your phone is compared against other users in real-time. When another user is nearby, both are prompted to take part in a duel where accelerometer data is used to determine the winner with points being applied to a global leaderboard. This app was designed to encourage spontaneous social interaction in the most unlikely places!
